DESKRIPSI
Sistem Rekrutmen PT. Mitra Data Abadi adalah aplikasi web berbasis Laravel yang dirancang untuk memudahkan proses rekrutmen karyawan dan penerimaan peserta magang. Sistem ini memungkinkan calon pelamar untuk mengajukan lamaran secara online, sementara tim HR dapat mengelola, mereview, dan memproses lamaran dengan efisien.
